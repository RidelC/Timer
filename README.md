# Timer
Timer pour la salle de classe, avec adpatation pour le sTiers temps avec trois modes.
# Temps flexible
Deux chronomètres, un pour le Temps standard, un pour le Tiers temps.
# Temps fixe
Un chronomètre quand on ne peut donner plus de temps aux élèves bénéficiant d'un tiers temps. En revanche onb peut définir de stâches différentes avec un minutage et un niveau mis à jour en temps réel.
# Double épreuve
C'est le mode Temps flexible doublé, dans le cas où deux épreuves se déroulent dans la même salle
